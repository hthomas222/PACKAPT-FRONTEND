PackApt

Purpose:

The purpose of this application is to auto-update the packages for Ubuntu. Lets say that the admin does not want to update every package, this application allows for the user to install 1 package, all of the packages, and it ensures that the packages are installed.


How to:

To begin, click the Package Compare button below this section. This will allow the end user to get the baseline of want packages are needed to be installed. The view button will display the results for the user to see. The end user can also update the packages as well. To do this, the end user must click the Update button in the navbar and this will allow the user to update a single or all of the packages.
